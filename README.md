## Getting Started

Follow these instructions to get the app running locally.

### Installation

 To install dependencies and seed database, run these commands in your local terminal while pointed to Film-Finder folder.
* npm
  ```sh
  npm i & npm run seed
  ```

  ```sh
  npm run develop
  ```
